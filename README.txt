このプログラムについて
----------------------
    Windows上で動くフリーソフトの、『U235』を真似て作って
  みました。まだβ版ですがとりあえず遊べます。

    このソフトウェアはフリーソフトウェアです。
  Windows/Linux/FreeBSD で動作するはずですが、保証は致しません。また、この
  プログラムによって起こったいかなる障害に対しても補償致しません。


起動方法
--------
  game.exeをダブルクリックするとフルスクリーンで起動します。
  debug.exeをダブルクリックするとウィンドウモードで起動します。
  *-nosound.batをダブルクリックするとサウンド無しで起動します。


操作方法
--------
    操作方法は、左右キーで画面下部の中性子砲（≦￥≧）を左右に動かし、
  左SHIFT キーで中性子（↑）発射、ウラン（＠）に中性子が当たると発電量が
  増えるので、発電量を競います。

  config.rbを編集することでパラメータを変更できます。
